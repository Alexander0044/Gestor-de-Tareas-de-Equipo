# Gestor de Tareas de Equipo

Aplicación de consola desarrollada en Java que permite gestionar una lista de tareas.

## Funcionalidades

- Añadir nuevas tareas
- Mostrar la lista de tareas registradas

## Estructura del proyecto

El proyecto está formado por tres clases principales:

- **Tarea.java**  
  Representa una tarea con una descripción y un estado de completado.

- **GestorTareas.java**  
  Contiene la lógica del programa y gestiona una lista de tareas mediante un `ArrayList`.

- **Main.java**  
  Contiene el menú interactivo que permite al usuario utilizar la aplicación.
